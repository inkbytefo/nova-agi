# NOVA (Non-Linear Optimization & Variational Architecture)

Nova is a research framework for AGI based on Hypergraph Neural Networks and Thermodynamic Learning.

## Structure

- `configs/`: Hydra configuration files.
- `src/nova/`: Source code.
  - `core/`: Low-level math & graph primitives (JAX).
  - `models/`: Neural network architectures.
  - `data/`: Data loaders & generators.
  - `utils/`: Utilities.
- `tests/`: Unit tests.
- `scripts/`: Training and evaluation scripts.

## Installation

```bash
pip install -r requirements.txt
```
